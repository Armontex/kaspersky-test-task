# План

pandas (или аналог типа openxls, ибо pd может жрать много памяти) - для составления таблиц
fastapi - для presentation
celery - для оркестрации процессами
rabbitmq - брокер сообщений
redis - для хранения мета-инфы о процессе решения задачи

пример обработки:

```mermaid


sequenceDiagram
    participant Client
    participant API as FastAPI
    participant RMQ as RabbitMQ
    participant C as Celery
    participant W as Worker
    participant R as Redis

    Client->>API: POST /public/report/export
    API->>API: Сохраняет файл в tmp/file_id
    API->>R: Status: PENDING (создана)
    API->>RMQ: Отправляет задачу (file_id=report_id)
    API->>Client: 202 Accept, report_id

    RMQ->>C: Доставка сообщения
    C->>W: Назначает задачу воркеру
    W->>R: Status: STARTED (в работе)
    
    W->>W: Проверка файла и обработка...
    
    alt Успешное выполнение
        W->>W: Создаёт reports/report_id.xlsx<br>удаляет tmp/file_id
        W->>R: Status: SUCCESS, path: ...
        W->>C: Task Done
        C->>RMQ: Удалить задачу из очереди
    else Ошибка
        alt Есть попытки
            W->>R: Status: RETRY (следующая через X сек)
            W->>C: Перезапуск
            C->>RMQ: Вернуть в очередь с задержкой
        else Попытки исчерпаны
            W->>R: Status: FAILED
            W->>C: Task Error
            C->>RMQ: Удалить проблемную задачу
        end
    end
```

пример получения файла:

```mermaid
sequenceDiagram
    participant C as Client
    participant API as FastAPI
    participant R as Redis

    C->>API: GET /public/report/report_id
    API->>R: Спрашивает информацию по report_id
    alt Есть инфа
        R->>API: Выдаёт status, report_path
        alt Status [PENDING, STARTED]
            API->>C: Возвращает статус в body
        else Status [SUCCESS]
            API->>API: Читает report_path
            API->>C: Отдаёт report.xlsx
        else Status [FAILED]
            API->>C: Сообщение об ошибке, предупреждение, что скоро данное сообщение будет удалено
        end
    else Нет инфы
        API->>C: Такой report_id не существует
    end
```
